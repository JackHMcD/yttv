# YTTV
Python API for YouTubeTV (https://youtube.com/tv)

## Installation
poetry install
wget https://github.com/gorhill/uBlock/releases/download/1.42.4/uBlock0_1.42.4.firefox.signed.xpi
poetry run python yttv.py
